Spotify Clone Backend API

This backend API provides a simple way to manage a Spotify-like application. It includes endpoints for managing artists, albums, tracks, playlists, and users. Below is the documentation for all available API endpoints.

Base URL

http://localhost:3000

Endpoints

Artists

1. Get All Artists

GET /artists

Response:

[
  {
    "id": "a1",
    "name": "Artist Name"
  },
  ...
]

2. Get Artist by ID

GET /artists/:id

Parameters:

  • id: The ID of the artist

Response:

{
  "id": "a1",
  "name": "Artist Name"
}

3. Create a New Artist

POST /artists

Request Body:

{
  "name": "New Artist Name"
}

Response:

{
  "id": "a2",
  "name": "New Artist Name"
}

4. Update Artist by ID

PUT /artists/:id

Parameters:

  • id: The ID of the artist

Request Body:

{
  "name": "Updated Artist Name"
}

Response:

{
  "id": "a1",
  "name": "Updated Artist Name"
}

5. Delete Artist by ID

DELETE /artists/:id

Parameters:

  • id: The ID of the artist

Response:

{
  "id": "a1",
  "name": "Artist Name"
}
